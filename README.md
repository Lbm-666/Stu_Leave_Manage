# 学生请假管理系统 - 接口自动化测试

## 项目简介

这是针对 **学生请假管理系统** 后端接口编写的自动化测试项目，使用 Python + `requests` + `unittest` 实现，包含 10 个测试用例，覆盖登录认证、请假申请、查询、审批等核心业务场景。

> **项目背景**：系统是一个 Spring Boot + Vue 的课程大作业，我独立完成后端开发和单元测试。为了深入学习测试，额外用 Python 编写了这套接口自动化测试。

---

## 测试覆盖

| 模块 | 测试场景 | 用例数 |
|------|---------|--------|
| 登录认证 | Token 鉴权机制 | 隐含在每个用例中 |
| 请假申请 | 正常申请、病假、长期请假、时间异常、空值验证 | 6 |
| 请假查询 | 学生查询自己的请假记录 | 1 |
| 审批流程 | 通过、拒绝、超3天流转 | 3 |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动后端服务

在 IDEA 中运行 `LeaveSystemApplication.java`，确保服务启动在 `http://localhost:8080`

### 3. 运行测试

```bash
python leave_api_test.py
```

## 后端接口说明

| 接口 | 方法 | 描述 |
|------|------|------|
| `/api/auth/login` | POST | 登录认证 |
| `/api/leave/apply` | POST | 提交请假申请 |
| `/api/leave/list` | GET | 查询请假记录 |
| `/api/leave/audit` | POST | 审批请假 |
